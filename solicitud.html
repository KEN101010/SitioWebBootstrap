Paso 2: El navegador realiza una petición

Después de que el usuario escribe una URL, el navegador genera una solicitud HTTP para pedir información al servidor.

HTTP significa HyperText Transfer Protocol y es el protocolo utilizado para intercambiar información en la Web.

Ejemplo visual

El navegador envía una solicitud al servidor para obtener la página web solicitada por el usuario.

Métodos HTTP más utilizados

Existen diferentes métodos HTTP. Los más comunes son GET y POST.

Método Descripción Cuándo se usa
GET Solicita información al servidor. Cuando se desea visualizar una página web.
POST Envía información al servidor. Cuando se completa un formulario.

Ejemplo de solicitud HTTP

Cuando el navegador solicita una página, puede enviar una petición similar a la siguiente: 

GET /index.html HTTP/1.1
Host: www.ejemplo.com
User-Agent: Navegador
Accept: text/html

Características de una solicitud HTTP

  • Se genera desde el navegador.
  • Viaja a través de Internet.
  • Contiene información sobre el recurso solicitado.
  • Permite la comunicación entre cliente y servidor.

📌 Dato importante

Cada vez que visitamos una página web, el navegador realiza una o varias solicitudes HTTP para obtener imágenes, textos y otros recursos.

¿Qué sucede después?

Una vez que la solicitud llega al servidor, este procesa la petición y prepara una respuesta para enviarla de regreso al navegador.
