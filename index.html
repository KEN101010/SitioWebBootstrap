Paso 1: El usuario escribe una URL

Todo comienza cuando un usuario escribe una dirección web, conocida como URL (Uniform Resource Locator), en la barra de direcciones de su navegador.

Ejemplo: https://www.google.com

La URL permite identificar el sitio web que el usuario desea visitar y es el punto de partida de toda la comunicación en la Web.

Ejemplo visual

El usuario escribe una URL, el navegador la interpreta y se prepara para enviar una solicitud al servidor correspondiente.

¿Qué contiene una URL?

Una URL está formada por varias partes que ayudan al navegador a localizar correctamente un recurso en Internet.

  • Protocolo: https://
  • Dominio: www.google.com
  • Ruta: /imagenes/foto.jpg
Componente Ejemplo Función
Protocolo https:// Define cómo se comunica el navegador con el servidor.
Dominio www.google.com Identifica el sitio web que se desea visitar.
Ruta /imagenes/foto.jpg Indica el recurso específico solicitado.

💡 Dato interesante

Los navegadores modernos recuerdan sitios visitados anteriormente para facilitar el acceso y mejorar la experiencia del usuario.

¿Qué sucede después?

Una vez que el usuario ingresa la URL, el navegador genera una solicitud HTTP para pedir la información al servidor donde se encuentra la página web.
